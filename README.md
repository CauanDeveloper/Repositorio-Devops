## 🥄 Receitas Git — Mise en Place

```bash
git init                    # abre o restaurante
git clone <url>             # traz a receita do repositório
git add .                   # coloca tudo na bancada
git commit -m "mensagem"    # finaliza o prato
git push origin main        # envia para o cliente
git pull origin main        # traz atualizações da cozinha
git status                  # confere o que está no preparo
git log --oneline           # ver todas as receitas anteriores
